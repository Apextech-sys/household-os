# HouseholdOS
AI-powered household intelligence platform. Upload documents, parse statements, track warranties, manage budgets — all powered by Claude AI.

## Stack
- Web: Next.js 14 App Router (PWA)
- Mobile: React Native / Expo
- Backend: Supabase (PostgreSQL + Auth + Storage + Realtime)
- AI: Anthropic Claude via Vercel AI SDK + OpenAI embeddings
- Deploy: Vercel (web) + Expo EAS (mobile)
